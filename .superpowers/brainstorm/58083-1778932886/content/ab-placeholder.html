交互流程说明

  1. 用户在配置面板新增分组 → 节点底部出现对应 Handle + 画布下方自动生成虚线占位框
  2. 用户从节点面板拖出一个节点，拖到占位框上方时占位框高亮（橙色发光）
  3. 松手 → 系统自动：创建节点、删除占位框、建立 AB_SPLIT → 新节点的连线（带分组标签）
  4. 用户也可以忽略占位框，直接从 Handle 拖线连接已有节点（占位框同步消失）
  5. 删除分组 → 对应占位框自动移除（若已连真实节点则断开连线并弹确认）

技术可行性

  • 占位框 = 特殊 ReactFlow 节点（type: ab-placeholder），保存时过滤掉
  • 拖节点到占位框：监听 onDrop，命中占位框 → 在其位置创建新节点并自动连线
  • 从 Handle 连线到占位框：onConnect 时检测 target 是占位框 → 重定向为真实连线
  • 分组增减同步占位框：在 onChange(bizConfig) 里对比新旧 groups，增删对应占位节点
