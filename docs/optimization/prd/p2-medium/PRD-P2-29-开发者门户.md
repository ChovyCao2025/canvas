## 1. 问题描述

### 1.1 现状

无 SDK 生成、无 API 发现工具。

### 1.2 痛点

- 开发者需要查阅 Swagger 文档才能调用 API
- 无法自动生成 SDK（SDKify）

---

## 2. 功能需求

### 2.1 核心功能

1. **SDK 自动生成**：
   - 基于目前 Swagger OpenAPI 规范
   - 支持 Java、Python、JavaScript SDK

2. **API 发现**：
   - API 索引页面（按模块归档：画布管理、CDP、审计）
   - 按版本索引（v1, v2）

3. **开发者文档**：
   - 快速上手指南
   - NodeHandler 开发指南
   - 示例代码包

### 2.2 工作量

| 任务 | 工作量（人天） |
|------|--------------|
| SDK 生成集成 | 5 |
| API 发现页面 | 3 |
| 文档集成 | 2 |
| 测试 + 文档 | 2 |
| **总计** | **12** |
