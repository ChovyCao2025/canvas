## 1. 问题描述

### 1.1 现状

发布画布时无法预估触发量级。

### 1.2 痛点

- 营销人员误触发百万量级画布执行，导致成本激增

---

## 2. 功能需求

### 2.1 核心功能

1. **受众量级估算工具**：
   ```
   输入：
   - 画布触达话术配置 → 预估触达率 (15%)
   - 目标人群 ID → 人群规模 (100,000)

   输出：
   - 预估执行量 (15,000)
   - 预估执行时间 (1.5 分钟)
   - 预估成本 (API 调用成本 $15)
   ```

2. **成本限制提醒**：
   - 如果预估执行量 > 系统配额（如 10,000）→ 触发「分发管控」审批

### 2.2 工作量

| 任务 | 工作量（人天） |
|------|--------------|
| 量级估算逻辑 | 3 |
| UI 预估面板 | 2 |
| 成本限制审批 | 2 |
| 测试 + 文档 | 1 |
| **总计** | **8** |
