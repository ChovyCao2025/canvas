## 1. 问题描述

### 1.1 现状

ADMIN 收到所有告警疲劳，非 ADMIN 收不到告警盲区。

### 1.2 痛点

- 部门负责人需要填写告警通知偏好（按阈值/按部门）但没有 UI
- 误操作（如误删画布）导致重要人员错过通知

---

## 2. 功能需求

### 2.1 核心功能

1. **通知偏好配置表** (`notification_preference`)：
   - `user_id`: 用户
   - `alert_level`: INFO/WARNING/CRITICAL
   - `channels`: ['email', 'push', 'sms']
   - `workgroup_id`: 通知群组

2. **默认规则**：
   - CRITICAL 级别：所有角色都通知
   - WARNING 级别：部门负责人接收

3. **偏好管理 UI**：
   - 用户个人设置「告警偏好」
   - 群级规则设置「部门关键成员推送」
