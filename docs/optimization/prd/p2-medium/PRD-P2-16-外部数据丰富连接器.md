## 1. 问题描述

### 1.1 现状

外部数据（如用户信用分、消费习惯）仅支持手动导入 CSV。

### 1.2 痛点

- 无法实时获取最新用户信用信息（如金融科技平台需要实时风控）

### 1.3 意义

对接第三方 API（如：DSU/超融智信）丰富用户画像，为 CDP 提供数据维度。

---

## 2. 功能需求

### 2.1 核心功能

1. **数据源配置**：添加外部数据源（API Key、API URL、请求模板）
2. **同步任务**：Cron 同步（每日/每小时/实时）
3. **数据更新策略**：遇字段冲突时「覆盖」或「合并」

### 2.2 工作量

| 任务 | 工作量（人天） |
|------|--------------|
| 数据源配置 UI | 3 |
| 同步任务引擎 | 3 |
| 数据一致性校验 | 2 |
| 测试 + 文档 | 2 |
| **总计** | **10** |
