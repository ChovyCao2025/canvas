## 1. 问题描述

### 1.1 现状

执行状态仅有 SUCCESS/FAILED，无法表示「部分成功」（如 3/5 分支成功）。

### 1.2 痛点

- 无法评估画布变更对大部分用户的影响（如更新节点影响率 60%）

---

## 2. 功能需求

### 2.1 核心功能

1. **部分成功状态**：`PARTIAL_SUCCESS`
2. **分支成功率统计**：
   - 按「成功/失败节点」分组
   - 统计各分支的平均成功率

### 2.2 工作量

| 任务 | 工作量（人天） |

| 测试 + 文档 | 1 |
| **总计** | **6** |
