## 1. 问题描述

### 1.1 现状

仅支持 Excel 标签导入，无法导入 CSV/JSON 文件和导出画布 JSON/PNG。

### 1.2 痛点

- 不支持数据科学团队使用 Python/CSV 格式导入导出画布数据
- 无法生成画布 PNG 用于文档演示

---

## 2. 功能需求

### 2.1 核心功能

1. **画布导出**：
   - PNG 图片（悬浮右键菜单导出）
   - JSON 格式（支持版本兼容）
   - YAML 格式（与 Git Sync 集成）

2. **标签导入扩展**：
   - CSV 格式支持（`user_id,tag1,tag2,tag3`）
   - JSON 格式支持（`[]` 数组结构）

3. **导出模板下载**：
   - 提供标准模板下载（避免格式错误）
