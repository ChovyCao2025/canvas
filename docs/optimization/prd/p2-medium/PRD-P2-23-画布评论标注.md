## 1. 问题描述

### 1.1 现状

团队讨论在系统外进行（企直通、钉钉、飞书群），画布内上下文丢失。

### 1.2 痛点

- 生产问题排查时需要查阅多个聊天记录
- 跨时区协作困难

---

## 2. 功能需求

### 2.1 核心功能

1. **画布注释**：
   - 在画布节点上直接标注评论
   - 支持可拖拽、stick（固定在屏幕侧边）

2. **线程回复**：
   - 每个注释支持子回复
   - @ping（提及同事）

3. **协作同步**：
   - 集成飞书 @提醒
   - 实时推送评论到企步 App
