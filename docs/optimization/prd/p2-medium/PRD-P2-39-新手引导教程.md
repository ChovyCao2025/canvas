## 1. 问题描述

### 1.2 痛点

## 2. 功能需求

### 2.1 核心功能

1. **引导流程**：
   - 首次登录 → 欢迎页 → 创建画布 → 添加节点 → 发布画布
   - 使用 `react-joyride` SDK 实现

2. **引导终止**：
   - 用户可跳过引导
   - 引导后终止（10 分钟内不再显示）
