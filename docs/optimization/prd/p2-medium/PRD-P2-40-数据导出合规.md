## 1. 问题描述

### 1.1 现状

无法导出执行日志供监管审查。

### 1.2 痛点

- 监管机构要求提供「个人数据删除证明」或「审计报告」

---

## 2. 功能需求

### 2.1 核心功能

1. **合规导出功能**：
   - 导出格式：XML（结构化）、CSV（脱敏）、PDF（人工签字）
   - 包含：
     - 执行时间范围
     - 失败节点列表
     - 用户 PII 脱敏数据
   - 导出时可选择「GDPR Art.17 脱敏」模式（删除敏感字段）

2. **按需导出申请**：
   - 管理员发起导出申请 → 审计员审核 → 审核通过后导出

| 任务 | 工作量（人天） |

| PII 脱敏导出 | 2 |

| 测试 + 文档 | 1 |
| **总计** | **8** |
