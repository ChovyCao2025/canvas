## 1. 问题描述

### 1.1 现状

无法在画布内直接邀请同事参与协作。

### 1.2 痛点

- 流程确认需要通过企步机器人消息逐个发送
- 无法在画布节点上分配责任人

---

## 2. 功能需求

### 2.1 核心功能

1. **画布内 @提及**：
   - 画布变量/会话消息支持 `@同事名` 自动转换为 `<mention:user_id>`
   - 选中的同事弹出侧边栏「申请编辑权」「申请查看权」

2. **节点责任人分配**：
   - 右键节点设置「负责人」
   - 列表视图显示每个节点的责任人

3. **协作邀请**：
   - 团队按钮邀请（弹出成员选择器）
   - 协作规则：「部门负责人可见，成员可编辑」

### 2.2 工作量

| 任务 | 工作量（人天） |
|------|--------------|
| 规则引擎（@提及应用） | 3 |
| 责任人 UI | 3 |
| 协作邀请 UI | 3 |
| 测试 + 文档 | 2 |
| **总计** | **11** |
