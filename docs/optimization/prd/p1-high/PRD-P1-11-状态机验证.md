## 1. 问题描述

当前允许发布 KILLED 画布和归档 DRAFT 画布，违反状态流转语义，可能导致数据不一致或越权操作。

---

## 2. 目标与价值

### 2.1 用户故事

> 作为 平台管理者，我希望 状态流转严格符合语义，禁止非法状态转换，以便 防止异常流程和越权操作。

### 2.2 成功指标

- 发布画布前校验状态为 DRAFT
- 归档前校验状态为 RUNNING

---

## 3. 功能需求

### 3.1 核心功能

1. **状态流转校验** - 发布画布前校验状态为 DRAFT
2. **归档校验** - 归档前校验状态为 RUNNING
3. **API 限制** - 禁止手动设置非法状态
