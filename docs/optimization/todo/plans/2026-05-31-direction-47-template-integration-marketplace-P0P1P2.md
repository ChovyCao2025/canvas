# 方向㊼：模板与集成市场(Marketplace) — 功能清单

> 定位：从"封闭平台"升级为"开放生态平台"——画布模板市场+节点插件市场+渠道连接器市场+素材市场+第三方开发者平台
> 策略评估：Marketplace是SaaS平台从"工具"到"生态"的关键一步。HubSpot App Marketplace(1000+ App)、Shopify App Store(8000+ App)、Slack App Directory均验证了Marketplace模式
> 竞品对标：HubSpot App Marketplace、Shopify App Store、Mirakl(企业级Marketplace)、Sharetribe(无代码Marketplace)、Origami Marketplace(B2B/B2C/C2C)
> 建议：**P2/远期**，依赖⑪开放平台+㊲开发者门户+⑮资源中心成熟后启动，Marketplace是生态化的"最后一公里"

## 现状盘点

| 功能 | 实现程度 | 差距 |
|------|----------|------|
| 画布模板 | **不存在** | 无模板库/模板分享/模板上架 |
| 节点插件 | **不存在** | 60+内置节点，无第三方节点扩展机制 |
| 连接器 | **不存在** | 渠道/数据源集成均为内置，无第三方连接器 |
| 开发者生态 | **不存在** | 无第三方开发者入驻/上架/收益分成 |

## 功能清单

### P0 — 模板市场 [1.5人月]

| 子功能 | 描述 |
|--------|------|
| 画布模板库 | 模板浏览(按行业/场景/渠道分类)+预览+一键导入+评分+评论 |
| 模板上架 | 用户/开发者创建模板→提交审核→上架→免费/付费 |
| 模板版本管理 | 模板更新+版本历史+兼容性标注 |
| 模板分析 | 模板下载量/使用量/评分/收入排行 |

### P1 — 插件与连接器市场 [2.0人月]

| 子功能 | 描述 |
|--------|------|
| 节点插件市场 | 第三方开发者创建自定义节点类型→上架Marketplace→用户安装 |
| 渠道连接器市场 | 新渠道连接器(如WhatsApp Business/LINE/KakaoTalk)→上架→安装 |
| 数据源连接器 | 新数据源连接器(如Snowflake/HubSpot/Salesforce)→上架→安装 |
| 插件SDK | 标准化插件开发SDK+文档+沙箱测试环境 |

### P2 — 生态运营 [1.0人月]

| 子功能 | 描述 |
|--------|------|
| 开发者入驻 | 开发者注册+资质审核+开发文档+API Key+收益仪表盘 |
| 收益分成 | 上架产品→收入分成(平台30%/开发者70%)+结算+提现 |
| 生态运营 | 推荐位/排行榜/编辑精选/认证徽章/促销活动 |
