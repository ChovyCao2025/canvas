# 方向㊹：品牌社区平台 — 功能清单

> 定位：从"品牌单向触达用户"升级为"用户之间的社区互动"——品牌社区+论坛+用户组+问答+倡导者计划+社区驱动营销
> 策略评估：CX Today/Innoloft/BuddyBoss 2026报告验证Community-Led Growth是B2B和DTC的增长核心战略，社区从"nice-to-have"变为"revenue infrastructure"
> 竞品对标：Salesforce Experience Cloud(CRM数据联动社区)、Khoros(企业级品牌社区)、Bettermode(现代UI+无代码)、Circle(创作者社区)、Mighty Networks(会员社区)
> 建议：**P2建议做**，依赖⑯协作权限+㉓对话式营销+㊱反馈调研成熟后启动，社区是品牌私域的"社交基础设施"

## 功能清单

### P0 — 社区基础 [2.0人月]

| 子功能 | 描述 |
|--------|------|
| 论坛与讨论区 | 分类/帖子/回复/点赞/最佳答案/搜索 |
| 问答(Q&A) | 提问→回答→投票→采纳答案→知识沉淀 |
| 用户组 | 按兴趣/产品/地区/会员等级创建私密/公开组 |
| 社区管理 | 版主权限+内容审核+垃圾过滤+敏感词+举报处理 |

### P1 — 倡导者与增长 [1.5人月]

| 子功能 | 描述 |
|--------|------|
| 倡导者计划 | 识别高活跃用户→授予倡导者身份+专属权益+任务 |
| 积分与等级 | 社区活跃→积分+等级(新手/成员/专家/大使)→权益差异化 |
| 社区内容→营销 | 优质UGC→自动推送到⑮资源中心→复用于Campaign |
| 社区活动 | 线上活动/AMA/产品反馈会/用户大会→活动管理+报名+回放 |

### P2 — 社区数据驱动 [1.0人月]

| 子功能 | 描述 |
|--------|------|
| 社区分析 | 活跃度/留存率/内容产出/支持偏转/社区ROI |
| 社区→画像 | 社区行为→写入用户画像(兴趣/专业度/影响力) |
| 社区触发画布 | 用户首次发帖→欢迎旅程; 成为倡导者→专属Campaign |
