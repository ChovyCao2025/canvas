# Architecture Archive

This folder contains source documents that were processed into the active architecture todo queue.

## Categories

- `reference/`: stable architecture references and summaries.
- `reviews/`: prior review reports, risk reports, and production checklists.
- `remediation/`: original remediation index and issue parts.
- `evolution/`: long-term target architecture and migration documents.

The archive is retained for traceability. Do not use it as the active implementation queue; use [../todo/README.md](../todo/README.md) instead.
