# Release Post Drafts

This directory contains public announcement drafts for the Open Source Growth
track. Drafts must stay aligned with the current gate state.

Rules for release drafts:

- Do not claim backend public extension/API write operations are stable before
  G10 passes.
- Do not claim production runtime plugin marketplace readiness.
- Do not claim a finalized license until the coordinator confirms it.
- Prefer local demo, mock provider, template, DSL, and docs claims that can be
  verified from the repository.
