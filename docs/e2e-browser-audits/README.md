# Module Review + Browser E2E Reports

每个 Codex 窗口只写自己的模块报告。

报告命名：

- `public-auth.md`
- `home-ops-collaboration.md`
- `canvas-journey.md`
- `marketing-suite.md`
- `bi-analytics.md`
- `cdp-audience.md`
- `admin-config.md`
- `tenant-admin.md`

报告必须同时包含：

- code review findings
- Browser E2E route progress
- bugs found
- bugs fixed
- verification commands and results
- Browser regression result
- blocked routes
- needs coordination
- remaining risks

不要把多个模块的结果写进同一个报告文件。
